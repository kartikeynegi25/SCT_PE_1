# Task 01 – Writing Better Prompts

## Objective

The objective of this task is to understand how prompt engineering techniques improve AI-generated responses. By refining vague prompts into clear and structured instructions, we can produce more accurate, relevant, and useful outputs.

---

## Skills Demonstrated

- Prompt Engineering
- Prompt Refinement
- Context Injection
- Output Formatting
- AI Communication

---

## Methodology

For each example:

1. Start with a vague prompt.
2. Identify its weaknesses.
3. Rewrite it with clear instructions.
4. Compare the AI responses.
5. Analyze the improvements.

---

## Examples Included

- Climate Change
- Professional Email
- Artificial Intelligence

---

## Conclusion

Small improvements in prompt wording significantly improve response quality. Adding context, audience, constraints, and formatting instructions makes AI outputs more reliable and easier to use.
